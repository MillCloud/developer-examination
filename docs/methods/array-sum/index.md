# 数组元素求和

传入一个数组 `array`，数组内所有元素均为 `number` 类型。返回数组内所有元素之和。

示例：

```text
输入：array = [1, 2, 3, 4, 5]
输出：15
解释：1 + 2 + 3 + 4 + 5 = 15
```
